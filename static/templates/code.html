I prefer to make and deal with open source. Here are a few solo projects I've worked on that have source code available.

In early 2014 I wanted to explore building large-scale web applications with Go and Cassandra. To help manage mapping Go data structures to column families, I developed ibis.

This is an early-stage work in progress. So far ibis provides higher-level helper routines for working with gocql and reflection automagic to reduce boilerplate.

While working at reddit in 2012, I set up an instance of etsy's statsd service, to collect live stats from all our app servers. This service aggregates stats on a periodic basis and stores them in graphite. Unfortunately, as a node.js service, statsd couldn't make good use of our multi-core CPU. It wasn't long before we topped out the amount of data we could collect.

I then wrote a new implementation in Python. Python was a more familiar language for me, and I could spread the work across the entire CPU with the multiprocessing module. I called this rewrite "tallier", and it was able to collect roughly three times as much data as before.

This served us well for about a year, until once again we started running into the limit of our processing power. Rather than spend more money for beefier EC2 instances, I decided to try writing it in Go. This was a fun and appropriate exercise for learning Go. It didn't take long before I had a complete replacement that was yet again several times faster. With some optimizations and the release of Go 1.2, we were able to collect all the data we wanted with plenty of capacity to spare.

When I got my first Android phone in late 2008, I wanted to take a stab at mobile development. I had recently enjoyed some falling sand toys on the web, so I decided to make one for Android.

In 2002 my friends and I were unhappy with the state of the art in posting funny quotes from IRC. Existing sites had moderators as gatekeepers. They didn't seem to have much effect on quality, and they were never around to approve submissions anyway. I decided to make my own, replacing moderators and ranking with a pretty effective recommender system.

That original implementation is no longer live, but in 2008 I reimplemented it on Google App Engine. All the data has survived, although the personalized ranking features no longer exist.
